Observations:

Reaction time increases as the time interval between display of two characters  increases from 0 second to 1 second.

You will observe that the reaction time for the case K,k (name match) will be more than for the case K, K( visual and name match )for a fixed time interval. This is because identical letters are judged by their visual characteristics while letters having the same name but different visual characteristics are compared by their verbal characteristics. Thus the K, K match was made on the basis of physical( visual code ) which is faster.

The difference between the reaction time for case  t(K,k) - t(K,K) should ideally be more for the first time interval  .i.e.0 second then for the other time intervals. If letters are shown after a time interval of 1 or 2 seconds the name coding takes over the visual coding and hence now the difference between reaction time for K, K and K, k is not so more.( verify  it  in  the  graph )
