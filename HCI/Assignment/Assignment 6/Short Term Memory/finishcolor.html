Observations

The graph shows two lines. Green color line represents timings for color match case in different intervals.

Reaction time increases as the time interval between display of two colors increases from 0 seconds to 1 seconds. It is obvious because the user has to first perceive , then think and react.

If you observe that for a particular time interval say 0.5 seconds if reaction times for all cases is almost same( all points coincide on the graph ) then we may conclude that you are predicting the next outcome and hence reacting faster. This is called learning effect.
